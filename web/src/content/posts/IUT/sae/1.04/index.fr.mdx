## Contexte

Projet réalisé durant le premier semestre du BUT Informatique, centré sur la conception et l'implémentation de bases de données avec PostgreSQL.

## Objectif

Développement d'une base de données relationnelle pour analyser les données statistiques du Titanic, avec accent sur :
- Conception d'un schéma de base de données normalisé
- Implémentation de requêtes SQL complexes
- Analyse et visualisation de données statistiques

## Réalisation

Le projet s'est déroulé en trois phases :

### 1. Conception de la base de données

- Analyse et spécification des besoins
- Création des diagrammes entité-association
- Définition des règles de gestion
- Normalisation du schéma

### 2. Développement

- Implémentation sous <a href={frontmatter.links.postgresql}>PostgreSQL</a>
- Import et validation des données
- Optimisation des requêtes
- Analyse statistique

### 3. Analyse des données

- Développement de requêtes SQL complexes
- Extraction des données statistiques
- Visualisation via <a href={frontmatter.links.raw}>RAWGraphs</a>

## Résultats

Le projet a abouti à :
- Une base de données relationnelle normalisée
- Un ensemble de requêtes SQL optimisées pour l'analyse statistique
- Des visualisations démontrant les résultats clés
- Une documentation complète des choix de conception et d'implémentation
